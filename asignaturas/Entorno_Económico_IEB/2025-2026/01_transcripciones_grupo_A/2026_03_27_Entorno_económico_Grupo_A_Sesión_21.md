# 1. Introducción: aprender a trabajar con información

La sesión introduce una idea clave:

> el valor no está solo en saber cosas, sino en saber trabajar con ellas

Se presentan dos herramientas:

- Obsidian → gestión de conocimiento
- Antigravity → interacción con IA

---

# 2. Dos funciones complementarias

## 2.1. Obsidian

Sirve para:

- almacenar información
- organizar conocimiento
- construir un sistema propio

Idea clave:

> es una extensión de la memoria

---

## 2.2. Antigravity

Permite:

- interactuar con modelos de IA
- generar contenido
- automatizar procesos

Idea clave:

> es una extensión de la capacidad de acción

---

# 3. Concepto fundamental: carpeta y archivo

Se introduce la base de cualquier sistema digital:

## 3.1. Carpeta

- estructura
- organización

## 3.2. Archivo

- contenido
- información

Diferencia clave:

- archivo → tiene extensión (.md, .txt, .py…)
- carpeta → no

# 4. Alfabetización digital

Se plantea una crítica:

> el sistema educativo tradicional no enseña a trabajar con tecnología real

Ejemplos:

- se enseña a escribir cartas
- pero no a gestionar archivos o datos

Idea clave:

> saber usar herramientas digitales es una habilidad básica

---

# 5. Tipos de archivos

Se mencionan ejemplos relevantes:

- `.md` → texto estructurado
- `.txt` → texto plano
- `.csv` → datos tabulares
- `.json` → datos estructurados
- `.py` → código

# 6. Memoria y sistema

Se introduce una analogía clave:

- batería → energía
- RAM → capacidad de trabajo

Idea clave:

> igual que sentimos la batería, deberíamos “sentir” la memoria

# 7. Demostración práctica con IA

4. generar contenido

- necesidad de herramientas

# 12. Cambio en la forma de enseñar

- el profesor transmitía conocimiento

Ahora:

- el profesor construye conocimiento en directo

# 13. Realidad compleja

Se introduce una idea importante:

> el mundo real no es limpio ni ordenado

Ejemplos:

- crimen

# 14. Economía “visible” vs “invisible”

Analogía:

- bancos centrales → sistema circulatorio

# 15. Migración y economía real

# 16. Experiencia personal: sistema penitenciario

- la realidad es distinta de lo que se percibe desde fuera
- existen prejuicios generalizados

Idea clave:

> la experiencia directa cambia la comprensión

# 17. Prejuicios y realidad

Se plantea:

> tendemos a simplificar problemas complejos

Ejemplo:

- criminalidad
- pobreza
- exclusión

---

# 18. Torre vs realidad

Se rompe una idea común:

> la sociedad no es una jerarquía clara (arriba/abajo)

Sino:

- trayectorias cambiantes
- fragilidad
- movilidad real

---

# 19. Ética vs ley

Se introduce una distinción clave:

- ley → normas formales
- ética → comportamiento real

Idea clave:

> cuanto menos funciona la ley, más importante es la ética

---

# 20. Idea final

> entender el entorno económico exige mirar más allá de los datos

Incluye:

- personas
- contextos
- sistemas invisibles
