# 4. Filosofía del código abierto

Concepto central de la sesión:

> Código abierto (open source).

Definición práctica:
- Código hecho por la comunidad
- Accesible
- Modificable
- Mejorable

Ejemplos:
- GitHub
- Bitcoin
- Ethereum
- Wikipedia

Contraste:
- Sistema cerrado → pago + dependencia
- Sistema abierto → libertad + responsabilidad

Mensaje implícito:
> Entender open source es entender el nuevo entorno económico digital.
