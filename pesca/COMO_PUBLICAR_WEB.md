# CÓMO PUBLICAR TU WEB "SPANISH PREDATOR" EN INTERNET

Ahora mismo tu web vive solo en tu carpeta. Para tener el dominio `spanishpredator.com`, sigue estos pasos. Es más fácil de lo que parece.

## PASO 1: Subir la web gratis a la nube (Netlify)
Vamos a usar un servicio profesional y gratuito llamado **Netlify** para poner tu web online en segundos.

1.  Entra en **[Netlify Drop](https://app.netlify.com/drop)** (no hace falta registrarse para probar, pero sí para mantenerla, regístrate con tu email, es gratis).
2.  Abre la carpeta de tu ordenador donde tienes los archivos (`.../Laboratorio`).
3.  **Arrastra** toda esa carpeta dentro del cuadro de la página web de Netlify Drop.
4.  ¡Listo! En unos segundos te dará un enlace raro tipo `wondrous-bagel-123456.netlify.app`. Tu web ya es mundial.

## PASO 2: Ponerle el nombre "SpanishPredator"
1.  En el panel de Netlify, pulsa en **"Site settings"** > **"Change site name"**.
2.  Pon **`spanishpredator`**.
3.  Ahora tu web será: **`https://spanishpredator.netlify.app`**.
    *   Este nombre es gratis y tuyo para siempre.

## PASO 3: El dominio ".com" (Opcional y de pago)
Si quieres estrictamente **`spanishpredator.com`** (sin el "netlify"), eso es un producto premium que hay que comprar (cuesta unos 12€ al año).

1.  En el mismo panel de Netlify, ve a **"Domain Management"**.
2.  Dale a **"Add custom domain"**.
3.  Escribe `spanishpredator.com`.
4.  Te dejará comprarlo directamente allí.

---

### RESUMEN
1.  Arrastra tu carpeta a [Netlify](https://app.netlify.com/drop).
2.  Cámbiale el nombre a `spanishpredator`.
3.  ¡Disfruta de tu web!
